# Trust Score System

A MERN stack application that detects fake and trusted reviews.

## Features

- User Authentication
- JWT Authorization
- Protected Routes
- Add/Delete Reviews
- Fake Review Detection
- Dashboard System
- Responsive UI

## Tech Stack

### Frontend
- React
- React Router DOM
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- JWT
- bcryptjs

## Installation

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
